## 📋 方法 A: 一鍵腳本(推薦)

### 步驟 1: 執行自動化腳本

1. 下載 `deploy_step1.bat` 到 `C:\Users\acer\gui-sorter-bot\v2`
2. 雙擊執行 `deploy_step1.bat`
3. 它會自動幫你:
   - ✅ 初始化 Git
   - ✅ 建立 .gitignore
   - ✅ Commit 所有檔案

### 步驟 2: 建立 GitHub Repo

1. 打開 https://github.com/new
2. Repository name: `gui-sorter-render`
3. 選 **Private**
4. 點 **Create repository**

### 步驟 3: 推送程式碼

GitHub 會給你兩行指令,類似:
```bash
git remote add origin https://github.com/你的帳號/gui-sorter-render.git
git push -u origin main
```

**在 `v2` 資料夾的 PowerShell/CMD 執行這兩行**

## 📋 方法 B: 手動指令

如果自動腳本有問題,手動執行:

```bash
# 1. 進入資料夾
cd C:\Users\acer\gui-sorter-bot\v2

# 2. 建立 .gitignore
echo .env > .gitignore

echo venv/ >> .gitignore

# 3. Git 初始化
git init
git add .
git commit -m "Initial commit"

# 4. 連接 GitHub (先在網頁建立 repo)
git remote add origin https://github.com/你的帳號/gui-sorter-render.git
git push -u origin main
```

---

## 🌐 步驟 4: 部署到 Render

### 4.1 登入 Render
1. 打開 https://render.com
2. 用 GitHub 登入

### 4.2 建立 Web Service
1. 點「New +」→「Web Service」
2. 連接 `gui-sorter-render` repo
3. 設定:
   - Name: `gui-sorter-bot`
   - Region: `Singapore`
   - Branch: `main`
   - Runtime: `Python 3`
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python bot_render.py`
   - Instance Type: `Free`

### 4.3 設定環境變數

點「Environment」,加入:

```
TELEGRAM_BOT_TOKEN = <你的 Telegram Bot Token>
ANTHROPIC_API_KEY = <你的 sk-ant-xxx>
NOTION_TOKEN = <你的 Notion Integration Token>
NOTION_DATABASE_ID = <你的 32 位 Database ID>
```

### 4.4 部署!
點「Create Web Service」,等 3-5 分鐘

## 🆘 遇到問題?

**GitHub push 失敗**:
- 確認已登入 GitHub
- 確認 repo 名稱正確
- 試試 `git push -f origin main`

**Render 部署失敗**:
- 看 Logs
- 檢查環境變數是否都設定了
- 確認 requirements.txt 存在

**Bot 沒反應**:
- 等 30 秒(冷啟動)
- 檢查 Render Logs
- 確認 TELEGRAM_BOT_TOKEN 正確
