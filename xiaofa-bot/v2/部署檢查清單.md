## 部署步驟(勾選完成)

### GitHub 設定
- [ ] 建立新資料夾 `gui-sorter-render`
- [ ] 放入 `bot_render.py`
- [ ] 放入 `requirements.txt`
- [ ] 放入 `render.yaml`
- [ ] 建立 `.gitignore`
- [ ] `git init` 初始化
- [ ] 在 GitHub 建立 private repo
- [ ] `git push` 推送程式碼

### Render 設定
- [ ] 用 GitHub 登入 Render
- [ ] 建立新 Web Service
- [ ] 連接到你的 repo
- [ ] 設定 4 個環境變數:
  - [ ] TELEGRAM_BOT_TOKEN
  - [ ] ANTHROPIC_API_KEY
  - [ ] NOTION_TOKEN
  - [ ] NOTION_DATABASE_ID
- [ ] 點「Create Web Service」
- [ ] 等待部署完成(3-5 分鐘)
- [ ] 複製你的固定 URL

### 測試
- [ ] Telegram bot 傳連結測試
- [ ] 確認 Notion 有新資料
- [ ] (選用)用 curl 測試 Flask API

### iOS 捷徑
- [ ] 打開捷徑 App
- [ ] 建立 3 個動作
- [ ] 填入你的 Render URL
- [ ] 測試分享連結

### 保持醒著(選用)
- [ ] 註冊 Uptime Robot
- [ ] 設定 5 分鐘監控
